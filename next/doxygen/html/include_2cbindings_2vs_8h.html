Detailed Description

Entrypoint for the C interface of libvs

Author
karurochari

libvs is meant to be used via its C++ interface, and only a subset of its functionality is exposed via these bindings.
Basically, what is needed to handle ui_trees, access to the constrained interface ui widgets have, resolving paths, symbols and little more.
